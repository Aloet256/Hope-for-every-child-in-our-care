# Kiddie Care Website

A complete responsive multi-page static website built with HTML, CSS and JavaScript.

## Open locally
Open `index.html` in a browser, or run a local server:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Pages
Home, About, Programs, Impact, Gallery, Get Involved, Donate, Safeguarding, Contact and Privacy.

## Before publishing
1. Add the organization’s verified phone, email and office details.
2. Connect contact, volunteer, newsletter and donation pledge forms to a secure backend/form service. The current static demo stores submissions in browser localStorage.
3. Add verified donation/payment details through a secure provider.
4. Review and formally approve the safeguarding and privacy text.
5. Confirm consent and usage rights for every photograph.

## Brand assets
The supplied logo has been cropped and converted to a transparent PNG at `assets/images/kiddie-care-logo.png`. All supplied photos are optimized as WebP files.
